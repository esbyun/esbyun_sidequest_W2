## Project Title

Sidequest Week 2: Emotional Blob

---

## Description

This project lets users move a blob around using keyboard inputs. It showcases the emotion of frustration.

The user will see a blob, which can be moved around using the keyboard: left/right or A/D to move side to side, up/W/Space to jump. Inspired by the emotion frustration.

---

## Interaction Instructions

Press A/D or left/right to move and up/space/W to jump.
